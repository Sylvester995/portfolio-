Cash Register App


Overview

The Cash Register App is a simple, user-friendly application designed to handle basic cash transactions in a retail environment. The app calculates the change to be returned to the customer based on the price of the item, the amount of cash provided by the customer, and the cash available in the cash drawer. It also provides appropriate messages to the user under various scenarios, ensuring smooth and efficient transactions.


Features

- Calculates the change to be returned to the customer
- Provides appropriate messages to the user based on the transaction
- Ensures smooth and efficient cash transactions
- Easy-to-use interface for retail employees
- Responsive design for use on various devices
- Cash Drawer Management: Keeps track of the cash available in the drawer.

Pokémon Search App


Overview

Roman Numeracal converter


Overview

The Roman Numeral Converter App is a simple yet powerful tool designed to convert numbers between Roman numerals and Arabic numerals. This app is perfect for anyone who needs to quickly and accurately perform these conversions, whether for educational purposes, historical research, or personal interest.


Features

- Convert Arabic to Roman: Allows users to input an Arabic numeral (e.g., 1999) and get the corresponding Roman numeral (e.g., MCMXCIX).
- Convert Roman to Arabic: Allows users to input a Roman numeral (e.g., MCMXCIX) and get the corresponding Arabic numeral (e.g., 1999).
- Validation: Ensures that the input provided by the user is valid for conversion.
- User-friendly Interface: Intuitive and easy-to-use interface for a seamless user experience.
- Error Handling: Provides appropriate messages to the user in case of invalid input or errors.

Music Player


Overview

This project is a feature-rich music player that includes functionalities like play/pause, next, shuffle, and stop, providing an intuitive and dynamic user experience. It fetches music data dynamically from an external API, demonstrating seamless integration of external services into web applications.


Features

- Play/Pause: Start or pause the currently selected track with a single click.
- Next/Previous Track: Navigate through the playlist effortlessly.
- Stop: Completely stop playback and reset the progress bar.
- Shuffle: Randomize the order of tracks for a unique listening experience.
- Dynamic Data Fetching: Fetch music data from an external API for real-time updates.
- User-friendly Interface: Intuitive design for a seamless user experience.
